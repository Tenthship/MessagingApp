# CHATADORI

["Chatadori is a real time web chat application built using Websockets"]

## Live Demo
https://message-app-pnrw.onrender.com/

## Screenshot
![App screenshot](ChatadoriScreenshot.png)

## Features
- Real-time messaging using WebSocket technology
- Live user count display
- Message timestamps
- Custom usernames
- Auto-scroll to latest messages
- Input sanitization for security
- Clean, modern UI

## Tech Stack
- **Backend:** Node.js, Express, WebSocket (ws library)
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Render

## Running Locally

1. Clone the repository
```bash
git clone [your-repo-url]
cd [your-repo-name]
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
npm start
```

4. Open browser to `http://localhost:3000`
